BLAST Help

What is BLAST?

BLAST (Basic Local Alignment Search Tool) is a fast sequence alignment tool used to find similarity between a query sequence and sequences in a database.

proISDB uses local BLAST+ to compare against IS insertion sequence elements in the database.

BLAST Types
  • blastn - Nucleotide sequence alignment (DNA sequences)
  • blastp - Protein sequence alignment (amino acid sequences)
Parameter Explanation
E-value Threshold
Expectation value, represents the probability of finding similar matches by chance in a random database.
Recommended: 10 (standard), 0.01 (strict)
Lower E-value = more significant match
Max Results
Maximum number of matches to return
Recommended: 50-100
Word Size
Minimum number of consecutive bases for initial match
11 - Standard (recommended)
7 - Short sequences or low similarity
15-20 - High similarity sequences
Sequence Format Examples
FASTA Format (Recommended):
>My_Query_Sequence

CGATCGATCGATCGATCGATCGATCGATCGATCGAT
Plain Sequence:

Result Interpretation
Alignment Score
Higher value indicates better match quality
Identity
Percentage of perfectly matched bases/amino acids
Coverage
Percentage of alignment range in query and target sequences
E-value Color
Green - Highly significant (<0.001)
Yellow - Significant (0.001-1)
Gray - Weakly significant (>1)
