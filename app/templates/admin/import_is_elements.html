Template specification
  • The first row must be column headers.
  • Required column: name (IS element name). Only name is required for admin imports — the current admin user will be recorded as the submitter automatically.
  • Optional columns: other columns listed in the template (e.g. family, group, mge_type, host, accession_number, synomyns, iso, origin, is_length, orf1, orf_1, comment, references, etc.).
  • Header names are lower-cased during validation; use the template for exact column names.
  • Do not include submitter/reviewer/status/timestamp columns (for example: submitter_first_name, submitter_last_name, submitter_id, reviewer_id, submission_date, review_date, review_comment, status, created_at, updated_at) — these will be set automatically to the current admin and system values during import.
  • CSV encoding should be UTF-8.
